Bidra till Helenelund.org

För att kunna se till att alla scheman hålls uppdaterade behöver vi DIN hjälp.

Skicka din klass schema till oss och få ditt namm/signatur på hemsidan!

Gör så här:

  • Öppna Skola24 på din dator, mobil eller surfplatta
  • Välj domänen "Sollentuna.skola24.se"
  • Logga in med E-legitimation(BankID)
  • Välj din klass schema
  • Skicka PDF:en till bidra@helenelund.org och skriv klassens namn i titeln. För att få ditt namn med på hemsidan krävs också ett namn eller signatur.

Tack!
