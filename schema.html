Bidra till Helenelund.org

För att kunna se till att alla scheman hålls uppdaterade behöver vi DIN hjälp.

Skicka din klass schema till oss och få ditt namn/signatur på hemsidan!

Med dessa sex steg kan du hjälpa både dig själv och många andra Helenelund.org-användare.

Gör så här:

  1. Öppna sollentuna.skola24.se på din dator, mobil eller surfplatta
  2. Klicka på "Logga in med E-legitiamtion" och logga in med valfri metod(BankID rekommenderas)
  3. Öppna schemavisaren som finns på Skola24:s startsida
  4. Klicka på texten "Visa mer" högt upp till höger och skriv sedan in vilken klass du går i. Var noga med att välja "Vecka" och inte bara en dags schema
  5. Klicka på de tre prickarna ovanför "Visa mindre" och välj "skriv ut/spara som PDF". Klicka på knappen "Hämta PDF"
  6. Slutligen skicka PDF:en till bidra@helenelund.org och skriv klassens namn i titeln. För att få ditt namn med på hemsidan krävs också ett namn eller signatur.
